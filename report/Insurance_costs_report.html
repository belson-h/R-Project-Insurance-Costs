Introduction

This report presents an analysis of insurance cost data with the aim of identifying which factors are most strongly associated with individual insurance charges. The dataset contains information on customer demographics, lifestyle factors, health-related variables, and insurance history.

The purpose of this analysis is twofold: first, to explore and describe patterns in the data through statistical summaries and visualizations; and second, to build regression models that can help explain variation in insurance costs and potentially serve as a support tool in future pricing decisions.

The data set contains information about customer_id, age, sex, region, bmi, children, smoker, chronic_condition, exercise_level, plan_type, prior_accidents, prior_claims, annual_checkups and charges.

Method

The analysis was conducted in R and structured across three main stages: data preparation, descriptive analysis, and regression modeling.

Data Preparations

The data set was loaded using the readr package and initially inspected using glimpse() and summary() to understand its structure and variable types. Missing values were identified in three variables: bmi (28 missing), exercise_level (22 missing), and annual_checkups (20 missing). Missing values were handled individually with the planned regression analysis in mind. For each variable containing missing observations, a missingness indicator variable was created, followed by imputation. Missing values in bmi and annual_checkups were imputed using the median, while missing values in exercise_level were replaced with “Unknown”.

Further, categorical variables were standardized by trimming whitespace and applying consistent title case formatting, before being converted to factors. A new ordered variable, age_group, was created by binning age into five intervals (18–30, 31–40, 41–50, 51–60, 61–75) to support group-level comparisons in the descriptive analysis. In addition, a derived variable, risk_group, was created to capture combined risk based on prior insurance history. This variable combines prior_claims and prior_accidents into a single categorical measure. Policyholders with no prior claims or accidents were classified as Low risk, those with two or more claims or accidents as High risk, and all others as Medium risk.

Descriptive Analysis

Summary statistics for the target variable charges were computed, including mean, median, standard deviation, min, max, and interquartile range. A range of visualizations were also produced using ggplot2 to explore the distribution of charges and its relationship with key variables. In the result section, only the visuals showing the strongest predictors will be displayed and discussed. However, the other variables that are not among the strongest predictors on their own may still contribute to the regression models.

Regression analysis

To predict insurance charges, three linear regression models were built using lm(). The models were constructed progressively, starting simple and adding complexity stepwise, in order to evaluate whether additional variables and interactions improved predictive performance.

The descriptive analysis indicated that three variables had a particularly strong relationship with charges: smoker, chronic_condition, and age. These were included in all three models as the core predictors.

  • Model 1: includes only these three strong predictors and serves as a baseline for comparison.
  • Model 2: extends the baseline by adding bmi, risk_group, plan_type, and exercise_level. These variables were selected based on their theoretical relevance to insurance costs, and although their relationship with charges was weaker than the core predictors, the descriptive analysis indicated a positive association worth exploring further.
  • Model 3: builds on Model 2 by adding an interaction term between age and smoker, based on the hypothesis that the cost of smoking may increase with age. An interaction term lets the model capture that.

Variables such as sex, region, and annual_checkups were excluded to keep the model interpretable, and because their theoretical link to individual charges was considered weaker or more ambiguous.

Code 
model1 <- lm(charges ~ age + smoker + chronic_condition, data = costs)

model2 <- lm(charges ~ age + smoker + chronic_condition +
               bmi + risk_group + plan_type + exercise_level,
             data = costs)

model3 <- lm(charges ~ age * smoker + chronic_condition +
               bmi + risk_group + plan_type + exercise_level,
             data = costs)

Model comparison

The three models are compared against each other using adjusted R² and AIC (Akaike Information Criterion). Adjusted R² measures how well the model explains the variation in insurance charges, expressed as a value between 0 and 1 where a higher value indicates a better fit. Unlike regular R², it adjusts for the number of predictors, meaning it only increases if a new variable actually improves the model.

AIC balances how well the model is fitted against model complexity, where a lower value indicates a better model. It is particularly useful when comparing models of different sizes, as it penalises unnecessary complexity.

Adjusted R²: Model 1 explains about 66% of the variation in insurance charges using only the three strong predictors. Model 2 jumps to 74% by adding BMI, risk group, plan type and exercise level, which is a meaningful improvement. Model 3 stays at 74%, meaning the interaction term between age and smoker didn’t add any real explanatory value beyond model 2.

AIC: Model 1 has the highest AIC, confirming it’s the weakest fit. Model 2 scores only slightly lower at 20 192 and Model 3 scores just above at 20 194, so once again the interaction term in Model 3 didn’t improve the fit enough to justify it.

Overall conclusion: Model 2 is the best model. It explains noticeably more than Model 1, and the interaction between age and smoking that Model 3 tests turns out not to be necessary, at least not in this data set.

Model 2 explains 74% of the variation in insurance charges, and the overall model is highly significant (p < 0.001). All included predictors were statistically significant, suggesting that each variable contributes meaningfully to predicting charges.

The strongest individual predictor was smoking status, where smokers on average incur approximately 7 423 higher charges than non-smokers. Having a chronic condition adds around 3 791 to charges, and each additional year of age increases charges by approximately 71. BMI also shows a positive effect, with each unit increase adding around 159 to charges.

Regarding risk group, policyholders classified as Low or Medium risk have notably lower charges compared to the High risk reference group, with Low risk customers having charges around 2 618 less. Customers on a Premium plan have charges around approximately 1 484 more than the Basic plan, and those with a Standard plan around 670 more. Finally, exercise level shows that lower activity is associated with higher charges, with Low exercise adding around 1 488 compared to the High exercise group.

Note that in all interpretations are made on the basis that each coefficient reflects the effect of that variable while holding all others constant.

Limitations

  • Linear regression assumes a straight-line relationship — charges may not behave that way

  • Right-skewed charges can violate normality assumptions (visible in QQ-plot)

  • The model can only find correlations in historical data, not causation

  • Some variables were excluded for simplicity — a real model might include them

  • risk_group is a simplified derived variable that loses some nuance from the raw claims/accidents data
